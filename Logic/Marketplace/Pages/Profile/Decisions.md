# Profile Page — Decisions

| Decision | Rationale |
|---|---|
| **Inline edit mode** | Faster than navigating to separate settings page |
| **Wallet + history inline** | Single view for financial overview |
| **Address book managed here** | Centralized location for delivery addresses |
| **Logout in settings** | Standard UX pattern |
| **Auto-sync from Firebase** | Changes from other devices reflected in real-time |
