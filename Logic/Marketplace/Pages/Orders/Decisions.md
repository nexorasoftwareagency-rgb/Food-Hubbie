# Orders Page — Decisions

| Decision | Rationale |
|---|---|
| **localStorage cache** | Offline access to order history; instant load |
| **Tab filters (All/Active/Completed/Cancelled)** | Quick filtering without search overhead |
| **Reorder → outlet page** | Simple MVP; full cart pre-fill is future work |
| **Status badges** | At-a-glance order state identification |
