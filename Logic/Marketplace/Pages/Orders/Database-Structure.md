# Orders Page — Database Structure

Uses same `Order` schema as documented in root `04-Database-Structure.md`.

Fields used on this page:
- `id`, `outletName`, `status`, `total`, `createdAt`
- `items` (for order summary preview)
- `isReviewed` (to show/hide review prompt)
- `riderName`, `deliveryAddress` (for order detail expansion)
