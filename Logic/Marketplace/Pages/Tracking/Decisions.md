# Tracking Page — Decisions

| Decision | Rationale |
|---|---|
| **9-step visual pipeline** | Shows full journey from Placed to Delivered |
| **Status-specific icons + messages** | Clear visual communication per stage |
| **ETA hardcoded (35 min)** | No real-time delivery prediction API available |
| **Demo auto-advance** | Useful for development/preview; disabled for production |
| **Inline review modal** | Faster than navigating to a separate review page |
| **Rider info card** | Customer can contact rider directly |
