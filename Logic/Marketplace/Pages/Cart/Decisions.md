# Cart Page — Decisions

| Decision | Rationale |
|---|---|
| **₹499 free delivery threshold** | Zomato/Swiggy standard; encourages larger orders |
| **2% projected cashback** | Gamification — shows potential savings |
| **`FREESHIP` coupon type** | Explicit free delivery code alongside threshold |
| **Coupon validation on Cart page** | Immediate feedback before checkout |
| **Outlet context in header** | User knows which restaurant they're ordering from |
| **Quantity < 0 → remove item** | Graceful UX instead of error |
