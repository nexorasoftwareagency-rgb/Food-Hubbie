# Points / Gotchas: PartnersPage

- **No Firebase integration** — status changes reset on page refresh
- **No add-partner form** — new partners require direct Firebase manipulation
- **No partner detail/edit** beyond approve/reject
- **Contact phone has no WhatsApp link** — unlike Customers page which includes `wa.me` links
- **Status colors hardcoded inline**: pending `"#f59e0b"`, approved `"#22c55e"`, rejected `"#ef4444"`
