# Points / Gotchas: FeedbackPage

- **No Firebase integration** — `MOCK_FEEDBACK` constant
- **Star distribution is fixed** (not computed from actual feedback) — 5-star = 45%, 4-star = 30%, 3-star = 15%, 2-star = 7%, 1-star = 3%
- **No reply/respond** to feedback functionality
- **No flag/moderation** for inappropriate reviews
- **No filtering or sorting** of feedback cards
- **Dish badge** — orange pill showing which dish the feedback references
