# Broadcast Tab — Decisions

## Design Decisions
1. **Rate-limited sending** — Prevents accidental spam or abuse from admin panel
2. **Audience segmentation** — 5 targeting options for relevant notifications
3. **Category tagging** — Helps recipients understand notification purpose
4. **Optional image URL** — Rich notifications with promotional banners
5. **Broadcast history** — Last 20 broadcasts for reference and auditing
6. **Stats tracking** — Total sent + weekly count for usage monitoring
7. **Audit trail** — Every broadcast logged with admin identity
